# Usage
1. First install all the dependencies.
```bash
# ruby dependencies
bundle install


2. Open psql, create a `DB` named `network_development` and `rails db:migrate`.

3. Start the server
```
rails s
```
5. Now, open it in browser:
```localhost:3000
```
